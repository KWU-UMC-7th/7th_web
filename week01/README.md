## 🥨 PR 규칙

**1. PR title 형식 맞춰주세요**
  > 주차 - 닉네임/이름 워크북 과제 제출
  > 
  > ex) 1주차 - 만두/이은서 워크북 과제 제출

<br>

**2. original 저장소에서 만들어 둔 자신의 branch로 PR 보내주세요**
  - main 브랜치로 PR 날리지는 않는지 확인해주세요!

- 프로젝트 전에 git에 익숙해지기 위해 진행하는 과정이기 때문에, 완벽하지 않아도 괜찮습니다.
